## [1.0.0] - 2026-04-20

### Added
- Initial release
- Text Search API support with pagination
- Place Details API support
- Strongly typed DTOs (PlaceSearchResponse, PlaceSearchResult, PlaceDetails)
- Fluent API for overriding API keys per request
- Laravel 11, 12, and 13 support
- PHP 8.5+ requirement
- 100% test coverage with Pest
- PHPStan Level 5 static analysis
- GitHub Actions CI for tests, PHPStan, and Laravel Pint
